Acceptance Criteria

The "Advanced Analytics" section is added to the settings tab for each bundle app and the announcement bar app
Merchants can connect their Google Analytics account to BusyBuddy via the settings tab
Once connected, the Google data is populated on the Analytics tab
Merchants can disconnect their Google account via the settings tab
Merchants can update / switch their Google account via the settings tab
The Google data sections on the Analytics tab always render, even when no Google account is connected. These sections render with an empty state ("No Google Analytics data")

Backend API Endpoints Required

Endpoint Method Description
/api/analytics/google/status GET Check if Google Analytics is connected, returns account email and property info
/api/analytics/google/connect POST Initiate OAuth flow, returns Google OAuth URL for authentication
/api/analytics/google/disconnect POST Disconnect the Google Analytics account
/api/analytics/google/data GET Fetch Google Analytics data (sessions, pageviews, users, traffic sources, top pages)
/api/analytics/google/callback GET OAuth callback endpoint for Google authentication

Implementation Notes

Security
  • OAuth 2.0 for secure Google authentication
  • Tokens stored securely in database
  • API endpoints protected by session middleware
Data Sync
  • Data fetched on-demand when Analytics tab is viewed
  • Supports 7-day, 30-day, and 90-day time ranges
  • Graceful error handling with user feedback
